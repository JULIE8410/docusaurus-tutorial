## <Highlight color='#1C5AEC'>주의사항</Highlight>
### <Highlight color='#1C5AEC'>마우스 이벤트가 실행되지 않음</Highlight>
구현 중에 직면한 제한 때문에 다음 예제가 예상대로 작동하지 않고 `onMouseDown` 및 `onMouseUp` 이벤트 핸들러가 실행되지 않습니다:
```jsx
<Slider.Root
  onMouseDown={() => console.log('onMouseDown')}
  onMouseUp={() => console.log('onMouseUp')}
>
  …
</Slider.Root>
```
대신 포인터 이벤트를 사용하는 것을 권장합니다.(예:`onPointerDown`, `onPointerUp`). 
위의 제한과 상관없이 이러한 이벤트는 모든 포인터 입력 유형(마우스, 터치, 펜 등)에 대해 실행되므로 크로스 플랫폼/장치 처리에 더 적합합니다.
